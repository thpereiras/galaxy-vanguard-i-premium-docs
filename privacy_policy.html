Privacy Policy

Last Updated: February 8, 2026

General Information

Galaxy Vanguard I - Premium is an offline game developed by thpereiras that prioritizes user privacy. This document explains how we handle data in our application.

Data Collection

We do NOT collect personal or sensitive data directly. However, the game uses Firebase (Analytics and Crashlytics) to understand gameplay and improve stability. This Premium version does not display ads and does not include in-app purchases.

Local Storage

The application uses local device storage only for:

  • Game settings: Audio preferences, language, and controls
  • Player progress: Unlocked levels and high scores
  • Game statistics: Achievements and match statistics

All this data stays exclusively on your device and is not shared.

System Features

Screen Control

  • The game uses "keep screen active" functionality (wakelock) during matches to prevent the screen from turning off automatically
  • This functionality is only activated during gameplay and serves exclusively to improve user experience

Audio and Multimedia

  • The application plays sound effects and background music
  • Does not access microphone or record user audio

Permissions

The application does not request special Android permissions beyond those necessary for:

  • Audio playback
  • Local preference storage
  • Screen power control (keep active during games)

Analytics and Telemetry

Firebase Analytics

  • Game Analytics: We use Firebase Analytics to understand game usage patterns and improve user experience
  • Data Collected: Game events, level completion rates, session duration, and general gameplay statistics
  • Device Information: Device model, operating system version, app version, and country/region
  • No Personal Data: Firebase Analytics does not collect personally identifiable information
  • Data Usage: Analytics data helps us improve game balance, identify popular features, and fix issues

Firebase Crashlytics

  • Crash Reporting: We use Firebase Crashlytics to collect crash reports and diagnose stability issues
  • Data Collected: Crash logs, stack traces, device information, OS version, app version, and approximate time of the crash
  • No Personal Data: Crash reports are not intended to contain personally identifiable information
  • Data Usage: Helps us fix bugs and improve app stability

Third-Party Services

Firebase services may collect:

  • General device information (model, OS version, app version)
  • IP address for analytics and fraud prevention
  • Game events and session data (Firebase Analytics)
  • Crash reports and performance data (Firebase Crashlytics)

Note: This data is collected by third-party services (Firebase), not by us directly.

Services We Do NOT Use

  • We do not display ads
  • We do not offer in-app purchases
  • We do not integrate with social networks
  • There is no login or user accounts
  • We do not collect personal information like names, emails, or phone numbers
  • We do not use location services or access GPS data
  • We do not access contacts, camera, or other sensitive device features

Privacy Controls

Opt-Out Options

  • Local Data: You can clear app data or uninstall the app to remove locally stored progress and settings
  • Offline Use: The game can be played offline; when offline, analytics/crash reports cannot be sent
  • Device Controls: You may restrict network access for the app using your device settings (if available)

Children's Privacy

Our game is designed to be family-friendly. For users under 13:

  • No personal data is collected directly by our app
  • We do not knowingly collect personally identifiable information from children

Security

Locally saved data is protected by standard Android system security measures. Firebase services use industry-standard security practices. We do not store or transmit personal data ourselves.

Minors

The game is suitable for all ages. Since we do not collect personal data, there are no specific privacy concerns related to minors.

Changes to This Policy

We will notify about significant changes to this policy through app updates or announcements in the project repository.

Contact

For questions about this privacy policy:

  • GitHub: thpereiras
  • Email: galaxyvanguardgame​@​gmail​.​com
  • Repository: https://github.com/thpereiras/galaxy-vanguard-i-premium-docs

Data Deletion

To completely remove all game data:

  1. Uninstall the application
  2. Locally saved data will be automatically removed

This privacy policy was created to meet Google Play Store requirements and accurately reflects Galaxy Vanguard I - Premium's privacy practices.
